# Glosa v1 build archive

This directory preserves the autonomous v1 implementation records as historical evidence:

- `BUILD-PLAN.md` is the completed dependency-ordered execution checklist.
- `OVERNIGHT-LOG.md` is the detailed build and review narrative.
- `orchestrator-prompt.md` is the unattended execution procedure used for that build.

These files are frozen records, not current priorities or instructions to resume the old build loop.
Use the root [ROADMAP.md](../../../ROADMAP.md) for accepted product direction, the public
[Glosa Roadmap project](https://github.com/users/davebream/projects/5) for live status, and GitHub
issues for executable work.
